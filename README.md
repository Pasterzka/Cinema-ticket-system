# Cinema-ticket-system
A system for booking cinema tickets.


## Runing the data generator

Instal python enviroment 
```bash
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

You can clear database by runing program:
```bash
python clear_db.py
```

You can generate data by running program:
```bash
python generator.py
```
